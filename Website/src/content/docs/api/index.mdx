# Chatbot API Documentation

Welcome to the documentation for the Chatbot API! This guide provides comprehensive information about the API's functionalities, endpoints, and usage.

## Overview

The Chatbot API allows developers to integrate a powerful chatbot into their applications, enabling users to engage in dynamic conversations, retrieve chat history, and even obtain images based on specified sources.

## Getting Started

If you're new to the Chatbot API, start with the following key sections:

- [**Quick Start:**](./quick-start) Get up and running with the API quickly using step-by-step instructions.

## Endpoints

Explore the various endpoints provided by the Chatbot API:

- [**Chat Endpoint:**](./endpoints/chat) Engage in dynamic conversations with the chatbot.
- [**New Character Endpoint:**](./endpoints/new) Introduce a new character and retrieve an introduction.
- [**Image Retrieval Endpoint:**](./endpoints/images) Retrieve images based on specified sources.
- [**Chat History Endpoint:**](./endpoints/chat-history) Retrieve chat history for a specific user.
